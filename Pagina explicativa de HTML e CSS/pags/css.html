BACKGROUND-COLOR

Background-­color serve para colorir determinado campo com uma cor.

Valores:

  • Para dizer qual cor podemos usar é possivél usar quatro modos: Black(A cor em ingles), #ff0000(Hexadecimal),rgb(255, 0, 0)[R=red,g=green,b=blue ;{as três cores primarias.}],rgba(255,0,0,0.5)[a=trantarencia]!

Exemplo:

body{
  background-color:#ff0000;
}

TEXT-SHADOW

Text­-shadow serve para adicionar sombra a um texto.

Valores:

  • Os dois primerois valores é a distancia que se estende a sombra.
  • O primero valor quando positivo vai para direita e negativo para esquerda.
  • O segundo valor quando positivo vai para baixo e negativo para cima.
  • O terceiro valor é a cor da sombra.

Exemplo:

p{
  text-shadow: 2px 2px #ff0000
}

Texto com sombra

MARGIN

Margin serve para criar espaço em torno de elementos.

Valores:

  • O os valores podem ser colocados em px.
  • 50px 25px 10px 5px;(Margem superior é 50px, margem direita é 25px, margem inferior é 10px, margem esquerda é 5px)
  • 50px 25px 10px;(Margem superior é 50px, margem direita e esquerda são 25px, Margem inferior é 10px)
  • 50px 25px;(Margens superior e inferior são 50px, margem direita e esquerda são 25px)
  • 50px;(Todas as quatro margens são 50px)

Exemplo:

p{
	margin: 50px 25px 10px 5px;
	margin: 50px 25px 10px;
	margin: 50px 25px;
	margin: 50px;
	margin-top: 50px;
	margin-right: 25px;
	margin-bottom: 10px;
	margin-left: 25px;
}

Executado exemplo:

Texto 50px superior, 25px direita, 10px inferior, 5px esquerda.

Texto 50px superior, 25px direita e esquerda, 10px inferior.

Texto 50px superior e inferior, 25px direita esquerda.

Texto 50px todos os lados.

Texto 50px superior, 25px direita, 10px inferior, 25px esquerda.

padding

padding serve para gerara espaços em torno do conteúdo.

Valores:

  • O os valores podem ser colocados em px.
  • 50px 25px 10px 5px;(O valor superior é 50px, o valor direita é 25px, o valor inferior é 10px, o valor esquerda é 5px)
  • 50px 25px 10px;(O valor superior é 50px, o valor direita e esquerda são 25px, o valor inferior é 10px)
  • 50px 25px;(O valor superior e inferior são 50px, o valor direita e esquerda são 25px)
  • 50px;(Todas as quatro margens são 50px)

Exemplo:

div{
	padding: 50px 25px 10px 5px
}

BOX-SHADOW

box-shadow serve para colocar uma sombra em um elemento.

Valores:

  • É estruturado por cinco valores:
  • O primeiro quando positivo a sombra vai para direita e negativo para esquerda./10px
  • O segundo quando positivo a sombra vai para baixo e negativo para cima./5px
  • O terceiro é a transparencia da sombra quanto maior o valor mais transparente fica./7px
  • O quarto é o tamanho da sombra./0px
  • O quinto é a cor da sombra./#999999(Hexadecimal)

Exemplo:

nav{
	box-shadow: 0px 5px 10px 0px #ff0000;
}

Maxheight serve para definir a autura maxima de um elemento.

Exemplo:

	max-height: 100px;

DISPLAY

O display serve para dizer como um elemento é exibido.

Valores:

  • inline:Um elemento linha que fica do tamanho exato do elemento.
  • block:O elemento bloco, usa todo espaço para direira ou esquerda de sue elemento pai(O elemento que ele está dentro!).
  • flex:Serve para fazer os elementos dentro do elemento pai flutuaren.
  • inline-block:O inline-block se comporta como linha e bloco, ou seja .

Exemplo:

li{
	display:inline-block;
}
