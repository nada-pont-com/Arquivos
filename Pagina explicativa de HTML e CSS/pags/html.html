Hypertext Markup language - HTML

HTML é a sigua de Hypertext Markup language e significa em português Linguagem de Marcação de Hipertexto, é uma linguagem para desenvolver websites.

HTML

A tag <html> diz ao navegador que este é um documento HTML.</html>

Atributos: Não foram vistos atributos para essa tag.

Exemplo:

<html>Incío e fim de um documento HTML.</html>

INPUT

A tag <input> especifica um campo de entrada onde o usuário pode inserir dados.

Atributos:

  • Type:É usada para capturar pequenos textos, escolhas simples, múltiplas escolhas e arquivos também./valores:text;password;radio;checkbox;file;hidden;button;reset;submit.
  • Radio:Serve para única escolha;
  • Checkbox:Serve para mutiplas opcões;
  • File:Serve para anequisar um arquivo;
  • Hidden:É um comando invisível;
  • Button:É um botão;
  • Submit:O botão de enviar;
  • Reset:Serve para limpar.

  • Name:O nome do controle, que é enviado junto com os dados do formulário.
  • Value:O valor inicial do controle.
  • Placeholder: Serve para adicionar um texto demostrativo, que é exibida até antes do usuario inserir um lavor.

Exemplo:

<html>
  <body>
    <form>
      >p>Nome>/p>
      >input type="text" name="nome" placeholder="Digite seu nome!">
      <br>
      <p>Senha</p>
      <input type="password" name="senha">
      <br>
      <p>Nome 2</p>
      <input type="text" name="senha" value="Nome 2">
    </form>
  </body>
</html>

Exemplo sendo Executado:

Senha
